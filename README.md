# Sangam – Modern Matrimonial Platform

Sangam is a modern matrimonial web application designed to help individuals find meaningful and lasting relationships through verified profiles, intelligent matchmaking, and secure communication.

## Features

* User registration and profile creation
* Verified member profiles
* Smart matchmaking experience
* Search and filter profiles
* Success stories section
* Dashboard for managing account activities
* Notifications and messaging interface
* Interest management system
* Responsive design for desktop and mobile devices
* Modern UI built with React and TypeScript

## Tech Stack

* React
* TypeScript
* TanStack Router / TanStack Start
* CSS
* Bun / Node.js

## Installation

### Using npm

```bash
npm install
npm run dev
```

### Using Bun

```bash
bun install
bun run dev
```

## Build for Production

### npm

```bash
npm run build
```

### Bun

```bash
bun run build
```

or
